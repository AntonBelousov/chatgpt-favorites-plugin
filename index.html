ChatGPT Favorites

A browser extension for saving and quickly reopening favorites in ChatGPT.

Get the latest release Leave feedback

Early public version. Feedback is welcome.

What you can save

Messages

Save individual ChatGPT responses you want to return to. View them later with full formatting, or copy as plain text.

Chats

Bookmark entire conversations so you can jump back to them without scrolling through your history.

Projects

Keep your active projects pinned and accessible. One click to open them again.

Why use it

Important things in ChatGPT get buried fast. A useful answer from last week, a project you were working on, a conversation you need to revisit — they all disappear into the sidebar.

ChatGPT Favorites keeps useful messages, important chats, and active projects easy to find again. Everything is stored locally in your browser.

Install

Download the latest version from Releases. Unzip, open chrome://extensions, enable Developer mode, and load the unpacked folder.
