ChatGPT Favorites

A browser extension for saving and quickly reopening favorites in ChatGPT.

Get the latest release Leave feedback

Early public version. Feedback is welcome.

Short demo

What you can save

Messages

Save exact answers or responses you want to reopen later.

Chats

Save full conversations you want to revisit quickly.

Projects

Keep active projects pinned and easy to reopen.

Why it helps

ChatGPT gets cluttered fast. Useful answers, important chats, and active projects disappear into the sidebar. Search and history don't always surface what you need.

Favorites gives you a faster path back to the exact thing you're looking for.

Install

  1. Download the latest release and unzip it
  2. Open chrome://extensions
  3. Enable Developer mode
  4. Click "Load unpacked" and select the unzipped folder
