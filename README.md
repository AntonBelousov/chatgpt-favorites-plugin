# ChatGPT Favorites

A browser extension for saving and quickly reopening favorites in ChatGPT.

## What you can save

- messages
- chats
- projects

## Why use it

Important things in ChatGPT get buried fast.

ChatGPT Favorites helps you keep useful messages, important chats, and active projects easy to find again.
